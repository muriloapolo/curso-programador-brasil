Tags Semanticas

Por que usar?

As tags dizem para os buscadores melhorarem sua página no ranqueamento. Faz com que o seu site seja encontrado mais facilmente. São tags utilizadas em Boas práticas.

Algumas TAGS

Header - Serve para menus/Navegação em geral.

main - Só pode existir uma por página

section - article - Não Existe um concenso. Porém, tem uma regra semântica.

Como o nome já diz. section define uma sessão. Já o article - funciona como um delimitador de uso. Se um grupo de elementos é uma sessão, o article será o texto sobre aquela matéria. Caso dentro daquele texto tenha conteúdo para levar o usuário até outro lugar, ele poderá receber uma section sem problemas.
