ARRAYS

  • Os arrays são como prateleiras que recebem diferentes elementes.
  • Os arrays começam com índice Zero, caso não sejam criados índices.
  • O array tem em seu tamanho igual a quantidade de itens menos 1.
  • Arrays podem ter seus itens acessados por um índex
  • Podem ser lidos com for e while
  • Criamos um array com o uso de let nomeVariavel = new Array() ou let nomeVariavel = []
  • Arrays podem ser varridos com In e Of, retornando algumas singularidades.
